Javascript Compared to HTML and CSS

HtML is the foundation to web pages - and is the static layout of text. To style this text we use CSS - cascading style sheets which adds color, images, decorative font and border styles and so on. Then in comes Javascript which is a dynamic scripting language which lets us the developers create interactive web pages. When a web page loads javascript can manipulate and change and the HTML elements and objects, css properties and style that you see in those pages - such as buttons and text input fields, color - the list is endless.

Control Flow & Loops

We all make decisions based on conditions and comparisons in every day life such as preparing and kneading a good pizza dough. Lets assume you have all the ingredients at hand.

  • You ask yourself - if it is a good time of day to make pizza dough? You must consider the temperature & humidity of the room.
  • You use a thermometer to calculate the temperature. About 25° C is ideal. You can proceed at this point.
  • If not ideal conditions you warm utensils and flour above warm oven.
  • You prepare by measuring ingredients to desired weights. Put aside excess flour
  • Then you allow the yeast to activate in your measured tepid water - timing five minutes
  • ¾ of the sifted flour first goes in to a bowl. Reserve ¼ for later
  • You make a well in the center of the flour & add the measured yeast, cold water, oil & salt
  • Then you slowly combine the flour in, stirring from center outward
  • If you have a thick batter you turn out to a floured surface
  • If not, you add a little more flour & mix
  • You then set a timer to 8 minutes
  • Then you knead quickly - gradually adding the ¼ flour you reserved earlier
  • Timer goes off
  • You decide if dough is at required consistency - not sticky
  • If too sticky you set timer for another 2 minutes & knead
  • If ok roll dough in to a tight ball
  • Cover & leave dough to rest
  • Done

Accessing Data Differently

Both arrays and object literals are containers that store or hold collections of data but are accessed in different ways.

Accessing data from an array requires you to point at an item by way of an index. Each item has a unique index number. Much like a cataloged book on a shelf.

Accessing data from object literals requires you to point at named keys for the values. The object literals are like boxes in a boxed compartment - like a P.O post office box where individuals can access mail with a unique key.

Functions

Functions group a set of operations with a given name that other code can call on or invoke to perform. They are useful because you may want to perform an operation more than once. They perform actions and methods on data like the objects mentioned above. Functions group logic and encourage code reuse. Less code and less duplicate code is written.
